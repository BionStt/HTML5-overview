Current Status: Candidate Recommendation (CR)
Canonical URI: http://www.w3.org/TR/webrtc-identity
Abstract: This document defines a set of ECMAScript APIs in WebIDL to allow and application using WebRTC to assert an identity, and to mark media streams as only viewable by another identity. This specification is being developed in conjunction with a protocol specification developed by the IETF RTCWEB group.
