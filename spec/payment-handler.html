Current Status: Working Draft (WD)
Canonical URI: http://www.w3.org/TR/payment-handler
Abstract: The Payment Request API provides a standard way to initiate payment requests from Web pages and applications. User agents implementing that API prompt the user to select a way to handle the payment request, after which the user agent returns a payment response to the originating site. This specification defines capabilities that enable Web applications to handle payment requests.
