Current Status: Working Draft (WD)
Canonical URI: http://www.w3.org/TR/media-capabilities
Abstract: This specification intends to provide APIs to allow websites to make an optimal decision when picking media content for the user. The APIs will expose information about the decoding and encoding capabilities for a given format but also output capabilities to find the best match based on the device’s display.
