Current Status: Working Draft (WD)
Canonical URI: http://www.w3.org/TR/mst-content-hint
Abstract: This specification extends MediaStreamTrack to provide a media-content hint attribute. This optional hint permits MediaStreamTrack consumers such as PeerConnection (defined in WebRTC) or MediaRecorder (defined in Mediastream Recording) to encode or process track media with methods more appropriate to the type of content that is being consumed. Adding a media-content hint provides a way for a web application to help track consumers make more informed decision of what encoder parameters and processing algorithms to use on the consumed content.
