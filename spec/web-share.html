Current Status: Working Draft (WD)
Canonical URI: http://www.w3.org/TR/web-share
Abstract: This specification defines an API for sharing text, links and other content to an arbitrary destination of the user's choice. The available share targets are not specified here; they are provided by the user agent. They could, for example, be apps, websites or contacts.
