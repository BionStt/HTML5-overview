--- layout: page category: spec title: "HTML Microdata" ---
Current Status: Working Draft (WD)
Canonical URI: http://www.w3.org/TR/microdata
Abstract: This specification defines new HTML attributes to embed machine-readable data in HTML documents in a style similar to RDFa. It is compatible with JSON, and can be written in a style which is convertible to RDF, although two-way conversion is not lossless.
